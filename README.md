## 4. 구현한 기능
- 로그인

<img width="856" title="로그인" alt="image" src="https://github.com/soonyeon/RLab-back/assets/108646588/4e14a3f3-bb21-4837-8f32-d5edfdd492ad">

> 헤더의 로그인 버튼을 클릭하면 로그인 모달창이 뜨고 로그인이 헤더에 프로필 이미지와 로그아웃 버튼이 보여집니다



- 스터디 멤버관리, 스터디 관리

<img width="856" alt="image" src="https://github.com/soonyeon/RLab-back/assets/108646588/b0c76001-0ce3-451b-b109-f90bf4900104">

> 로그인한 유저가 스터디장인 스터디 목록이 보여지고 해당 스터디를 클릭하면 스터디 멤버리스트와 관리 버튼이 나타납니다


> 박스 상단의 스터디 관리 탭을 클릭하면 스터디 관리 페이지로 이동하며, 스터디수정, 완료, 삭제가 가능합니다
---
